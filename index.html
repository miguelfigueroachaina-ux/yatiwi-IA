🍽️ Gastronomía

  • Trucha del Titicaca — el plato más representativo
  • Chairo — sopa andina con chuño y carne
  • Caldo de cabeza — tradicional de las ferias
  • Quinua — grano sagrado de los Andes

🗣️ Idiomas del Altiplano

En Puno conviven el español, el quechua (Yachay = sabiduría) y el aymara (Yatiwi = conocimiento). Ambas lenguas son reconocidas oficialmente por el Estado peruano.

📅 Calendario Cultural

  • May: Cruz Velacuy
  • Jun: Inti Raymi en el altiplano
  • Nov: Aniversario de Puno y emergencia del Manco Cápac

🎯 Misión

Preservar y difundir el patrimonio cultural de Puno mediante Inteligencia Artificial, conectando la sabiduría ancestral con las nuevas generaciones.

⚙️ Tecnología

  • IA: Claude (Anthropic) via API
  • Frontend: HTML5 + PWA
  • Hosting: GitHub Pages
  • Instalable: Progressive Web App

👥 Equipo
